# Graph Traversal System (BFS & DFS)

## Project Description
This project implements a graph using adjacency list representation in Java. It supports Breadth-First Search (BFS) and Depth-First Search (DFS) traversal algorithms.

## Features
- Graph representation using adjacency list
- BFS traversal
- DFS traversal
- Performance measurement using System.nanoTime()
- Experiments on different graph sizes (10, 30, 100 vertices)

## Algorithms

### BFS (Breadth-First Search)
BFS explores the graph level by level using a queue.

### DFS (Depth-First Search)
DFS explores the graph deeply using recursion.

## Complexity
Both BFS and DFS have time complexity of:
O(V + E)

## Experiments
The program tests graph performance on:
- Small graph (10 vertices)
- Medium graph (30 vertices)
- Large graph (100 vertices)
